# Product Requirements Document

## Nombre

AyudaInvestigacion

## Visión

Convertirse en la herramienta gratuita de referencia para diseñar proyectos de investigación en español.

## Usuarios objetivo

* Estudiantes TFG
* Estudiantes TFM
* Doctorandos
* Residentes sanitarios
* Tutores académicos
* Investigadores noveles

## MVP v0.1

### Funciones

1. Elegir tipo de proyecto
2. Definir tema
3. Generar pregunta de investigación
4. Sugerir diseño metodológico
5. Crear objetivos generales y específicos
6. Checklist ético básico

## No incluye inicialmente

* Registro de usuarios
* Backend
* API de pago
* Chatbot libre

## Métricas de éxito

* Usuarios que completan flujo
* Tiempo medio de uso
* Satisfacción
* Descargas/exportaciones
