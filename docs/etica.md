# Guía ética básica para proyectos de investigación

## Checklist ético básico
- [ ] Riesgo-beneficio identificado y justificado.
- [ ] Población y criterios de inclusión/exclusión definidos.
- [ ] Plan de consentimiento informado (si aplica).
- [ ] Medidas de confidencialidad y seguridad de datos definidas.
- [ ] Justificación del acceso a datos clínicos/personales.
- [ ] Plan para manejo de incidencias y retirada de participantes.

## Protección de datos
- Minimiza el dato: recoge solo lo estrictamente necesario.
- Limita acceso a la información sensible.
- Define periodo de conservación y eliminación de datos.
- Documenta base jurídica y medidas de seguridad aplicables.

## Consentimiento informado
- Debe ser claro, comprensible y voluntario.
- Incluye finalidad, procedimiento, riesgos y beneficios esperados.
- Debe explicar derechos: retirada, acceso y aclaraciones.
- En población vulnerable, reforzar explicaciones y salvaguardas.

## Anonimización y pseudonimización
- **Anonimización**: elimina identificadores para evitar reidentificación razonable.
- **Pseudonimización**: reemplaza identificadores por códigos, manteniendo clave separada y protegida.
- Aplica controles técnicos y organizativos acordes al riesgo.

## Advertencia importante
Esta guía **no sustituye** la valoración formal de un CEI/CEIm, ni la supervisión de tutores, metodólogos o responsables institucionales.
