# Metodología básica para diseñar proyectos de investigación

## Principios básicos
1. **Claridad del problema**: define qué quieres estudiar y por qué es relevante.
2. **Pregunta delimitada**: especifica población, fenómeno/intervención y resultado.
3. **Coherencia interna**: pregunta, objetivos, diseño, variables y análisis deben estar alineados.
4. **Viabilidad**: evalúa tiempo, recursos, acceso a muestra y factibilidad ética.
5. **Rigor y transparencia**: documenta criterios, decisiones y limitaciones.

## Tipos de estudio y errores frecuentes (tabla resumen)

| Tipo de estudio | Elementos metodológicos clave | Error frecuente a evitar |
|---|---|---|
| Observacional transversal | Definir variable principal, covariables y corte temporal único. | Inferir causalidad a partir de asociación. |
| Cohorte | Exposición, resultado, seguimiento y control de confusión. | No planificar pérdidas de seguimiento. |
| Casos y controles | Definición rigurosa de caso/control y medición de exposición previa. | Selección de controles no comparables. |
| Ensayo clínico | Intervención, comparador, seguridad, tamaño muestral y plan analítico. | Omitir aspectos éticos reforzados o análisis por intención de tratar. |
| Cualitativo | Fenómeno, contexto, muestreo teórico y saturación. | Falta de justificación del proceso analítico y de reflexividad. |
| Revisión sistemática | Pregunta PICO/PEO, búsqueda reproducible, selección por pares y evaluación de calidad. | No seguir PRISMA o no registrar protocolo cuando corresponde. |
| Scoping review | Pregunta amplia, criterios PCC y mapeo de evidencia. | Presentar conclusiones de efectividad no sustentadas por el diseño. |
| Delphi | Panel experto, rondas, consenso predefinido y análisis de concordancia. | No definir umbrales de consenso antes de iniciar. |

## Errores frecuentes transversales
- Preguntas demasiado amplias o ambiguas.
- Variables no operativizadas.
- Diseño elegido sin justificar su idoneidad.
- Muestra insuficiente o no representativa.
- Ignorar sesgos y factores de confusión.
- Olvidar criterios éticos y protección de datos.

## Recomendación
