# AyudaInvestigacion

Herramienta gratuita para ayudar a estudiantes e investigadores noveles a diseñar proyectos de investigación académicos y sanitarios.

## Objetivo

Facilitar la creación de:

* TFG
* TFM
* Tesis doctorales
* Protocolos de investigación
* Estudios observacionales
* Revisiones sistemáticas

## Problema que resuelve

Muchos proyectos presentan errores frecuentes:

* preguntas mal formuladas
* objetivos poco claros
* diseños inadecuados
* variables mal definidas
* aspectos éticos incompletos

## Solución

Asistente guiado paso a paso, gratuito y accesible desde navegador.

## Estado actual
